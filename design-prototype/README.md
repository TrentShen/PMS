## 使用方式

所有文件均为纯 HTML + 内联 CSS + 内联 SVG，无需任何外部资源或网络请求。

1. 在文件管理器中双击任意 `.html` 文件即可用浏览器打开预览。
2. 页面间通过左侧边栏导航互相链接（使用相对路径 `./xxx.html`）。
3. 桌面端为完整侧边栏 + 顶部全局栏布局；移动端顶部栏会显示菜单按钮，点击可展开侧边栏抽屉。
4. 预览响应式效果：在浏览器中按 `F12` 打开开发者工具，使用「设备模拟（Device Mode）」或拖拽窗口边界，将视口分别切换到 1280px、768px、375px 观察布局变化。在 1024px 以下侧边栏会变为抽屉；在 768px 以下表格会卡片化、KPI 可横向滑动、长表单页面出现底部固定操作栏。

## 响应式断点

| 断点 | 宽度范围 | 布局策略 |
|------|---------|---------|
| `xl` | `>= 1280px` | 完整左侧边栏（240px）+ 顶部栏 + 内容区居中，最大宽度 1440px |
| `lg` | `1024px - 1279px` | 完整左侧边栏，多列网格保持，内容区自适应 |
| `md` | `768px - 1023px` | 侧边栏隐藏为抽屉，顶部栏显示汉堡菜单；网格从 4/3 列压缩为 2 列 |
| `sm` | `640px - 767px` | 单列布局，表格转卡片列表，KPI 卡横向滑动，底部固定操作栏 |
| `xs` | `< 640px` | 紧凑单列，更小的卡片间距，触控目标放大至 44-48px |

> 实现说明：`styles.css` 中使用 `max-width: 1023px`、`max-width: 767px`、`max-width: 639px` 与 `min-width: 1280px` 四个媒体查询，分别对应 1024 / 768 / 640 / 1280px 断点。

## 桌面端布局特点

- **高信息密度**：表格行高 48px，KPI 卡 4 列，表单左右分栏，长表单左侧步骤条 + 右侧内容。
- **多栏视图**：HR 看板左侧筛选器 + 右侧 KPI/图表；上级评估左侧下属列表 + 右侧详情面板；绩效校准时顶部分布图 + 下方矩阵。
- **常驻操作**：关键按钮始终可见，批量操作工具栏常驻表格上方。
- **横向空间利用**：内容区最大宽度 1440px，居中显示，避免大屏幕上拉伸变形。

## 移动端布局特点

- **抽屉侧栏**：侧边栏从左侧滑出，点击外部或切换页面可关闭；顶部栏左侧显示汉堡菜单。
- **卡片化列表**：管理型表格（HR 管理台、校准矩阵、部门概览）在小屏下转为 `table-card-list`，每张卡片显示 3-4 个关键字段，操作按钮放在卡片底部。
- **横向滑动**：KPI 指标卡使用 `kpi-grid-scroll`，在小屏下可横向滑动浏览，避免一屏堆叠过多。
- **底部固定操作栏**：长表单页面（自评、上级评估、校准）使用 `form-bottom-actions`，将“保存 / 提交 / 退回”固定在屏幕底部，方便单手操作。
- **触控目标放大**：按钮、图标按钮、下拉框高度调整为 44-48px，符合移动端可触规范。
- **步骤条简化**：自评页使用 `steps-compact` 只显示当前步骤名，减少横向占用。

## 新增的关键响应式类

| 类名 | 作用 |
|------|------|
| `.table-card-list` | 移动端表格卡片化容器，默认隐藏，小屏下显示为 flex 列 |
| `.table-card` / `.table-card-header` / `.table-card-body` / `.table-card-footer` | 单条卡片结构：头部、2 列/1 列字段、底部操作 |
| `.kpi-grid-scroll` | 移动端 KPI 横向滚动容器，内部 `.kpi-card` 固定宽度 |
| `.form-bottom-actions` | 底部固定操作栏，移动端显示，带安全区适配 |
| `.mobile-only` / `.desktop-only` | 控制元素在移动/桌面端显隐 |
| `.hide-on-mobile` / `.hide-on-tablet` / `.hide-on-desktop` | 按断点显隐工具类 |
| `.steps-compact` | 移动端步骤条简化版，替代 `.steps` |
| `.grid-2` / `.grid-3` / `.grid-4` | 桌面网格，断点下自动降级 |

- **绩效校准**：状态条、分布对比、人员矩阵、审批按钮；移动端分布图简化、人员卡片化、底部审批操作。
